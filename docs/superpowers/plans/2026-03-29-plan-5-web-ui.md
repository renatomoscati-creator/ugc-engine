# Plan 5: Web UI Pages — Dashboard, Library, Calendar, Analytics, Recommendations, Settings

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Build all remaining frontend pages. The app shell (root layout, sidebar, navigation) already exists from Plan 1. All pages are Server Components reading from the shared SQLite database via Drizzle. Mutations go through API routes added here or in prior plans.

**Architecture:** Next.js 16 App Router, Server Components for data fetching, shadcn/ui, Tailwind dark mode, Geist Sans.

**Depends on:** Plan 1 (db schema, Drizzle client, app shell), Plan 2 (scripts table), Plan 3 (assets, production_jobs tables), Plan 4 (schedules, automation config, recommendations tables)
